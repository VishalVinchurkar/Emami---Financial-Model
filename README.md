# Emami Limited Financial Modelling & Valuation

## Overview

This project is a financial modelling and valuation analysis of Emami Limited, a leading Indian FMCG company. It was developed as a learning project to apply financial modelling concepts and valuation techniques using Microsoft Excel.

The model includes historical financial statement analysis, financial forecasting, Discounted Cash Flow (DCF) valuation, Comparable Company Analysis (Comps), ratio analysis, and sensitivity analysis to estimate the company's intrinsic value.

---

## Project Objectives

- Build an integrated three-statement financial model.
- Forecast the company's financial performance based on historical trends and key assumptions.
- Estimate intrinsic equity value using DCF valuation.
- Compare valuation with listed industry peers using trading multiples.
- Analyze the company's financial performance through key financial ratios.
